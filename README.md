## GitHub Pages 博客

## 本地运行

安装 Ruby
```bash
winget search Ruby
winget install RubyInstallerTeam.RubyWithDevKit.3.4
```

进入项目主目录, 安装项目依赖
```bash
bundle install
```

启动服务器
```bash
bundle exec jekyll serve
```


预览效果: [http://localhost:4000](http://localhost:4000)
实际效果: [https://sherlocknieh.github.io](https://sherlocknieh.github.io)
